GetStarted

For new users, we recommend following steps for a quick start.

Step 1 Get an Authorization Token

For the use of these APIs, it is requested to require an authorization token in the Authorization header to authenticate. To get an authorization token, please send an email to maptrix@navinfo.com.

Step 2 Get Service APIs

Our APIs provide common services for all users and catalogs.

For more details about the Service APIs, see the API Reference.
