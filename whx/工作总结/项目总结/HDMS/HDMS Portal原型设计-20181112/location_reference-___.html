Location Reference

Overview

Content

NavInfo Stable Topology(ST) is a representation of the road network that models intersection-to-intersection(node that connects at least 3 links).

Application scenarios

Location Reference (ST Layer) cross refererences ADAS data/Dynamic data/OEM specific data of any supplier to the static map.

• STs are modified only when road and intersection connectivity changes and changes impact the modeling result.

• STs are allocated unique IDs that are stable across map versions, and across different location related products (for example: SD Map, HD Map, traffic, weather, etc.)

Customers need the ability to merge data from different providers (OEMs, 3rd Party or Tier-1 suppliers), data sets (e.g. ADAS, weather, traffic) and even map precision levels (e.g. SD, HD). 

NavInfo provides a Location Reference layer function to support the OEM applications need to seamlessly join the different content with a specific location (location ID, offset,  range) in the static HD Map. The Location Reference layer is called Stable Topology (ST Layer).
