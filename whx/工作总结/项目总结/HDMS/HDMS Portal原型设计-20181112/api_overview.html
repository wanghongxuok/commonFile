Overview

NavInfo HD Map services provide RESTful interface for data access and ingestion. Data consists of Map data, Campaign jobs, Sensor Data and the Feedback information. For the map data with NDS and Protobuf format, it is organized in a Partition-Layer-Version Based manner and provides navigation data, dynamic traffic information and OEM data defined by third party.
